🚀 Project Stack

Next.js (Frontend)
Node.js + Express (Backend)
Tailwind CSS
shadcn/ui
MongoDB 

▶️ How to Run

Frontend
cd frontend
npm install
npm run dev
Open: http://localhost:3000

Backend
cd backend
npm install
Create .env file
npm run dev
Runs on: http://localhost:3001

⚙️ Environment Variables
Frontend:
NEXT_PUBLIC_API_URL=http://localhost:3001/api

Backend:

MONGO_URL=' '
